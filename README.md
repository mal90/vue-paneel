# vue-paneel

> Simple dynamic navigation panel using vue2,bootstrap-vue,vue-router.
>http://vue-paneel.herokuapp.com


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
